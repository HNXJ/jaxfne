## Overview

This tutorial builds a small recurrent E/I network (9 excitatory, 3 inhibitory neurons) using the native jaxfne public API. The network receives constant native drive, is coupled through all-to-all signed weights (positive from E neurons, negative from I neurons), and is driven by white noise.

The simulation demonstrates:
1. How neuron-type signatures in the Izhikevich emitter determine recurrent coupling strength and polarity
2. How recurrent feedback shapes voltage dynamics and spike timing
3. How to extract multimodal proxy readouts from a single simulation
4. The scope boundaries of computational scaffolds: simulation is relative, not biophysically calibrated

---

## Key Concepts

### Recurrent Input Equation

$$I_i(t) = I_i^{base} + \sum_j W_{ij} s_j(t) + \xi_i(t)$$

- $I_i(t)$ = total current to neuron $i$ (uncalibrated proxy units)
- $I_i^{base}$ = native base drive (cell-type specific: E=5 pA proxy, I=3 pA proxy)
- $W_{ij}$ = recurrent weight from $j$ to $i$ (signed by type; positive if $j$ is excitatory, negative if inhibitory)
- $s_j(t)$ = spike output of neuron $j$ (binary at each timestep)
- $\xi_i(t)$ = Gaussian white noise (std 0.5 proxy units)

### Implementation

The recurrent weights are generated by `_default_eig_connectivity()` in `jaxfne/emitters.py`:

```python
def _default_eig_connectivity(sign: jax.Array, dtype: jnp.dtype) -> jax.Array:
    n = sign.shape[0]
    pre_sign = sign[None, :]
    weights = jnp.ones((n, n), dtype=dtype) * pre_sign
    weights = weights * (1.0 - jnp.eye(n, dtype=dtype)) / jnp.sqrt(jnp.maximum(1, n))
    return 0.5 * weights
```

This creates an all-to-all connectivity matrix with:
- **Polarity determined by presynaptic neuron type:** Excitatory presynaptic neurons contribute positive weights, inhibitory contribute negative
- **Normalized by 1/√n:** Scales with network size for stability
- **Zero diagonal:** No self-connections
- **Scaled by 0.5:** Conservative coupling strength

During simulation, recurrent input is computed as:

```python
syn = weights @ prev_spikes
current_native = drive + syn + noise
```

## Public API Used

```python
import jaxfne as jtfne

cfg = jtfne.Configuration()
cfg = cfg.runtime(seed=42, dtype="float32", duration_ms=1000.0, dt_ms=0.1)
cfg = cfg.column("small_recurrent_ei", layers=["L2/3"], n=12)
cfg = cfg.cell_types({"E": 0.75, "PV": 0.25})
cfg = cfg.connectivity(excitatory_to_inhibitory=True, inhibitory_to_excitatory=True)
cfg = cfg.set_emitter("izhikevich", "cortical_eig_e_plus_pv")
cfg = cfg.probes(["MUA-proxy", "source-proxy", "LFP-proxy", "CSD-proxy", "EEG-proxy", "MEG-proxy", "EMM-proxy"])

model = jtfne.construct(cfg)
signals = jtfne.simulate(model, duration_ms=1000.0, dt_ms=0.1, seed=42)
```

**No local simulator code or hand-written dynamics loops required.** All recurrent E/I coupling is implemented in the native `simulate_eig_izhikevich()` function.

## Observed Behavior

### Voltage Dynamics
- Excitatory neurons typically exhibit resting potential around –65 mV (reset value) and spike threshold at 30 mV
- Recurrent inhibitory input suppresses excitatory voltage upstroke
- Noise introduces trial-to-trial variability

### Spike Raster
- Shows the temporal pattern of spikes for each neuron
- Excitatory neurons (red) are driven by native excitatory drive and inhibited by inhibitory spikes
- Inhibitory neurons (blue) fire in response to excitatory input and provide feedback suppression

### Population Firing Rate
- Rolling-average firing rate may show periods of elevated activity and quiescence
- Depends on balance between excitatory drive and inhibitory suppression
- Deterministic given seed, so same network repeated produces identical raster

### Source Proxy
- Integrated population activity signal, proportional to total synaptic current
- Not a real extracellular potential; proxy only
