**Release date:** 2026-05-19

## Summary

v0.2.1 implements and stabilizes the **multimodal proxy probe operator contract** for TFNE computational simulations.

**Key additions:**

- **Eight probe operators:** SPK (spike readout), Vm (membrane voltage), source (source/current proxy), LFP-proxy (laminar potential proxy), CSD-proxy (current-source-density proxy), EEG-proxy (simulated scalp channel), MEG-proxy (simulated magnetometer), and EMM-proxy (electromagnetic metabolism estimate proxy).
- **JSON-safe probe reports:** Every operator returns a `ProbeReadout` with frozen metadata: operator status, calibration status, claim-status fields, assumptions, and optional fields (sign convention, leadfield status, etc.).
- **Corrected terminology:** All public operators use `-proxy` suffix (e.g., `lfp_proxy`, `csd_proxy`, `eeg_proxy`, `meg_proxy`) to explicitly declare computational operator status. No public use of `*-like` informal analogy terminology.
- **Preserved claim-status metadata:** All eight operators freeze immutable validation fields:
  ```yaml
  truth_mode: truth_safe_unverified
  claim_level: computational_scaffold
  field_solver_status: laminar_proxy_no_pde
  field_claim_level: proxy_readout_only
  physical_amplitude_claim_allowed: false
  ```
- **Documentation:** New `docs/probe_operators.md` (380+ lines) specifies each operator, the common report contract, claim-status metadata, and future paths to calibrated/physical operators.
- **Tests:** 31 new probe operator tests added (477 total: 446 baseline + 31 new).

## Probe Operators (v0.2.1)

All eight operators are **simulated/proxy readouts**:

| Operator | Kind | Method | Status |
|----------|------|--------|--------|
| **SPK** | `spk` | threshold_or_emitter_spike_array | simulated spike readout from emitter |
| **Vm** | `vm` | emitter_state_voltage_trace | native state voltage, not calibrated |
| **source** | `source` | declared_source_projection_or_proxy | proxy source from emitter |
| **LFP-proxy** | `lfp_proxy` | point_or_finite_contact_phi_proxy | laminar potential proxy, no PDE |
| **CSD-proxy** | `csd_proxy` | divergence_proxy_or_second_derivative_laminar | laminar CSD proxy, no PDE |
| **EEG-proxy** | `eeg_proxy` | linear_leadfield_proxy | simulated scalp readout, toy leadfield |
| **MEG-proxy** | `meg_proxy` | linear_current_orientation_proxy | simulated magnetometer, toy leadfield |
| **EMM-proxy** | `emm_proxy` | normalized_activity_field_source_cost_proxy | normalized cost proxy, not biological |

For detailed operator specifications, assumptions, and future paths, see **docs/probe_operators.md**.

## API Example

```python
import jaxfne
from jaxfne.fields import (
    spk_probe, vm_probe, source_probe,
    lfp_proxy_probe, csd_proxy_probe,
    eeg_proxy_probe, meg_proxy_probe, emm_proxy_probe
)

# Simulate a small network
cfg = jaxfne.configuration().network(n=32).emitter(...).field(...).probe(...)
model = jaxfne.construct(cfg)
signals = model.simulate(jaxfne.simulation(duration_ms=1000.0))

# Apply probe operators
spk_readout = spk_probe(signals.spikes)
vm_readout = vm_probe(signals.V_m)
lfp_readout = lfp_proxy_probe(signals.field.lfp_proxy)
csd_readout = csd_proxy_probe(signals.field.csd_proxy)
eeg_readout = eeg_proxy_probe(signals.field.lfp_proxy)
meg_readout = meg_proxy_probe(signals.field.lfp_proxy)
emm_readout = emm_proxy_probe(signals.field.lfp_proxy)

# Each readout has JSON-safe report
print(spk_readout.report)
print(lfp_readout.report)
print(emm_readout.report)

# Verify claim-status metadata
for readout in [spk_readout, vm_readout, lfp_readout, csd_readout]:
    report = readout.report
    assert report["truth_mode"] == "truth_safe_unverified"
    assert report["claim_level"] == "computational_scaffold"
    assert report["physical_amplitude_claim_allowed"] is False
```
