## Quick Start

```python
import jaxfne

# Configure a simple cortical column
cfg = (
    jaxfne.configuration()
    .network(
        name="V1_proxy",
        kind="cortical_column",
        n=100,
        layers=["L2/3", "L4", "L5", "L6"],
        cell_types={"E": 0.8, "PV": 0.1, "SST": 0.07, "VIP": 0.03},
    )
    .emitter(family="izhikevich", preset="cortical_eig")
    .field(
        domain="laminar_column",
        conductivity="proxy",
        boundary="mean_zero_neumann",
        gauge="mean_zero",
    )
    .probe(name="laminar_probe", modes=["spikes", "V_m", "CSD", "LFP"], n_contacts=16)
)

# Construct and simulate
model = jaxfne.construct(cfg)
sim = jaxfne.simulation(duration_ms=300.0, dt_ms=0.1, seed=0, record_sources=True, record_fields=True)
signals = model.simulate(sim)

# Compute readouts and manifest
readouts = model.compute_readout(
    signals,
    [
        jaxfne.readout_spec("rate", "spike_rate_hz"),
        jaxfne.readout_spec("source", "source_abs_mean"),
        jaxfne.readout_spec("lfp", "lfp_abs_mean"),
    ],
)

manifest = model.manifest(signals, readouts)
print("Manifest version:", manifest.get("manifest_schema_version"))
print("Field claim level:", manifest.get("field_claim_level"))
```

## Known Limitations

1. **Source Calibration:** Izhikevich parameters use native current units. No empirical calibration against real neural data.
2. **Field Solution:** Laminar proxy mode (no PDE solve). Uses resistive kernel approximation for CSD/LFP readout.
3. **Whole-Brain Simulation:** jaxfne targets laminar proxy workflows. Multi-area integration is scaffolded but not validated.
4. **Plasticity:** Native plasticity support is not yet implemented.
5. **Device Support:** JAX CPU-first. Multi-GPU scaling via pmap/pjit (experimental).
