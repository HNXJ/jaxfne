# jaxfne

**jaxfne** is a JAX-native framework for source-to-field/readout workflows in computational neurophysics. Build custom models with JAX, project them into extracellular fields, and extract multimodal readouts (spikes, voltage, LFP proxy, CSD proxy, EEG proxy, MEG proxy, EMM proxy) with validated metadata.

## Core workflow

Starting from model outputs—whether generated by your own emitters, [Jaxley](https://github.com/google/jaxley)-compatible models, or other JAX computations—jaxfne organizes them into a coherent pipeline:

- **Source tensors** — Spatially localized currents or emitter states from your model
- **Field/proxy operators** — Extracellular potential and current-source density proxy operators, applied to sources
- **Probe readouts** — Eight multimodal readout operators: spikes (SPK), voltage (Vm), source projection (source), LFP proxy, CSD proxy, EEG proxy, MEG proxy, EMM proxy
- **Output bundles** — JSON-serialized results with metadata, validation status, and asset hashes

## Technical capabilities

- **JAX-native execution** — vmap, jit, and scan-based computation for efficient batching and differentiability on CPU and GPU
- **Multimodal readouts** — Eight independent probe operators for simultaneous inspection of spiking, voltage, field proxies, and computed metrics
- **Jaxley compatibility** — Consume Jaxley-style compartmental models or standalone JAX/NumPy workflows
- **Calibration-ready** — Designed to support future empirical calibration workflows; current outputs are labeled as proxies, not physical claims
- **Reproducible outputs** — All results serialized as JSON-safe bundles with metadata, hashes, and scope declarations
- **Field/proxy diagnostics** — Explicit metadata for proxy mode vs. future physical-candidate workflows

## Quick start

Install from PyPI:
```bash
pip install jaxfne
```

Run a minimal single-neuron example:
```bash
python examples/03_single_neuron_multimodal_probe.py
```

Or explore the interactive Colab tutorial:
```bash
# Open in Colab: notebooks/01_single_neuron_multimodal.ipynb
# Or run locally: jupyter notebook notebooks/01_single_neuron_multimodal.ipynb
```
