# jaxfne

**jaxfne** is a JAX-based framework for source-to-field/readout workflows in computational neurophysics. Build custom models with JAX, project them into extracellular fields, and extract multimodal readouts (spikes, voltage, LFP proxy, CSD proxy, EEG proxy, MEG proxy, EMM proxy) with metadata.

## Core workflow

Starting from model outputs—whether generated by your own emitters, [Jaxley](https://github.com/google/jaxley)-compatible models, or other JAX computations—jaxfne organizes them into a coherent pipeline:

- **Source tensors** — Spatially localized currents or emitter states from your model
- **Field/proxy operators** — Extracellular potential and current-source density proxy operators, applied to sources
- **Probe readouts** — Eight multimodal readout operators: spikes (SPK), voltage (Vm), source projection (source), LFP proxy, CSD proxy, EEG proxy, MEG proxy, EMM proxy
- **Output bundles** — JSON-serialized results with metadata, validation status, and asset hashes

## Technical capabilities

- **JAX-based execution** — vmap, jit, and scan-based computation for efficient batching and differentiability on CPU and GPU
- **Multimodal readouts** — Eight independent probe operators for simultaneous inspection of spiking, voltage, field proxies, and computed metrics
- **Jaxley compatibility** — Consume Jaxley-style compartmental models or standalone JAX/NumPy workflows
- **Calibration-ready** — Designed to support future empirical calibration workflows; current outputs are labeled as proxies
- **Reproducible outputs** — All results serialized as JSON-safe bundles with metadata, hashes, and scope declarations
- **Field/proxy diagnostics** — Explicit metadata for proxy mode vs. future physical-candidate workflows

## Quick start (v0.3.4)

Install from PyPI:
```bash
pip install -U "jaxfne>=0.3.5"
```

Minimal example with chainable grammar:
```python
import jaxfne as jtfne

cfg = jtfne.Configuration()
cfg = cfg.runtime(seed=7, dtype="float32", duration_ms=1000.0, dt_ms=0.1)
cfg = cfg.column("single_neuron", layers=["L2/3"], n=1)
cfg = cfg.cell_types({"E": 1.0})
cfg = cfg.connectivity()
cfg = cfg.set_emitter("izhikevich", "cortical_eig")
cfg = cfg.probes(["MUA-proxy", "source-proxy", "LFP-proxy"])

model = jtfne.construct(cfg)
signals = jtfne.simulate(model, duration_ms=1000.0, dt_ms=0.1, seed=7)
print(f"Spike count: {signals.spikes.sum():.0f}")
```
